# BaIAninho — Assistente Inteligente de Vendas e Estoque

> Sistema de apoio inteligente para **análise de vendas e controle de estoque**, desenvolvido com foco em **tomada de decisão estratégica**, utilizando **Inteligência Artificial**.

---

## Sobre o Projeto

O **BaIAninho** é um sistema de **auxílio automatizado para análise de dados de vendas e estoque**, desenvolvido para facilitar a gestão, reduzir erros humanos e acelerar a tomada de decisões empresariais.

Ele utiliza **Inteligência Artificial** para interpretar dados, gerar relatórios, responder perguntas em tempo real e oferecer insights estratégicos de forma simples e acessível.

O sistema possui **dois níveis de acesso**:
- **CEO (Administrador)** – acesso total ao sistema
- **Gerente** – acesso restrito apenas aos dados de vendas

## Objetivo do Sistema

- Automatizar a **análise de vendas**
- Facilitar o **controle de estoque**
- Gerar **relatórios inteligentes**
- Oferecer **respostas em tempo real** sobre os dados
- Auxiliar o **CEO e o Gerente** na tomada de decisões
- Reduzir tempo, erros e custos operacionais

---

## Funcionalidades Principais

✅ Leitura automática dos dados de vendas
✅ Análise de estoque, faturamento, entre outros.
✅ Produtos mais vendidos  
✅ Produtos com baixo estoque  
✅ Identificação de tendências  
✅ Relatórios automáticos  
✅ Chat com Inteligência Artificial  
✅ Separação de permissões por usuário  
✅ Sistema seguro de acesso  

---

## Níveis de Acesso

### CEO (Administrador)
Tem acesso total ao sistema:
- Vendas
- Estoque
- Relatórios completos
- Métricas financeiras

### Gerente
Tem acesso restrito:
- Apenas dados de **vendas**
- Faturamento
- Produtos mais vendidos
- Relatórios comerciais

> O gerente **não tem acesso** às informações completas da empresa.
